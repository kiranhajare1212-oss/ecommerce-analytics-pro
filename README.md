# 🚀 Advanced E-commerce Analytics Project

## 📊 Overview
This project showcases an end-to-end data analytics pipeline using SQL, Excel, Python, and Power BI.

---

## 🛠️ Tools & Technologies
- SQL (Data Analysis)
- Excel (Data Cleaning & Pivot Tables)
- Python (Pandas for automation)
- Power BI (Interactive Dashboard)

---

## 🔍 Key Features
- Revenue Analysis
- Customer Segmentation
- Customer Lifetime Value (CLV)
- Monthly Growth Trends
- Interactive Dashboard

---

## 📈 Dashboard Highlights
- KPI metrics (Revenue, Customers, Growth)
- Product & Category Analysis
- Customer Insights

---

## 🧠 Insights
- Top customers contribute majority revenue
- High-performing products drive business growth
- Seasonal patterns observed in sales

## ⚡ Outcome
Developed a complete analytics solution from raw data to interactive dashboards and automated analysis.
